# Animated Countdown Timer 

A simple animated countdown timer built with **HTML**, **CSS**, and **JavaScript**. The animation counts down from 3 to 0 and then shows a "GO" message with a replay button.

## Features

- Smooth countdown animation
- Replay button to restart countdown
- Minimalist and responsive design
